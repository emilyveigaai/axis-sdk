# AXIS SDK

**Version:** 0.3.0  
**Status:** Repository Migration (Phase 1)  
**License:** Proprietary - Enterprise Antigravity Labs

## Purpose

The `axis-sdk` is a **zero-dependency boundary layer** that defines contracts between AXIS components. It enables repository separation by providing stable interfaces that implementations can depend on without coupling to concrete logic.

## Installation

```bash
pip install axis-sdk
```
